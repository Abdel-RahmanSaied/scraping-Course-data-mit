
# scraping Course data from MIT

Python script to scraping data from MIT using :
- requests 
- BeatifulSoup 
- CSV 

and finally show it in data frame using :
- pandas


## Installation

Install Libraries with pip 

```bash
   -> pip install requests
   -> pip install beautifulsoup4
```
